# 📊 SIR Model Simulation

This project demonstrates how infectious diseases spread using the SIR (Susceptible–Infected–Recovered) model.

## 🧠 Concept
The population is divided into:
- S (Susceptible)
- I (Infected)
- R (Recovered)

The system is modeled using differential equations and solved numerically.

## ⚙️ Implementation
- Python (NumPy, SciPy, Matplotlib)
- Numerical solution of differential equations
- Graph-based visualization

## 📈 Results
The simulation shows:
- Rapid increase in infections
- Peak infection point
- Gradual recovery of population

## 📁 Project File
- `sir_model_simulation.ipynb`

## 🚀 How to Run
1. Open in Google Colab
2. Run all cells
3. View simulation graph
